Environment Installation
============

The web project relies on the Django framework (python). See this [web page](https://docs.djangoproject.com/en/1.8/intro/install/) for installation instructions. The project has been built on a 1.8 version of Django.

Run
===
First, you can adjust the project settings by copying the file `vinosite/local_settings.py.dist` to `vinosite/local_settings.py`.

For running the test server:
`python manage.py runserver`

See also this [django tutorial](https://docs.djangoproject.com/en/1.8/intro/tutorial01/) for basics on managing django application.
